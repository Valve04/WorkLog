ℹ️
วิธีเตรียม Notion ก่อนเชื่อม
1
ไปที่ notion.so/my-integrations → New Integration → ตั้งชื่อ Work Log → Submit
2
คัดลอก Internal Integration Token (ขึ้นต้นด้วย secret_...)
3
สร้าง Database ใน Notion พร้อม property: Name(Title), Work Type(Select), Role(Select), Team(Select), Staff Type(Select), Resource Name(Text), Date(Date), Days(Number), Hours(Number), Remarks(Text)
4
เปิด Database → ... → Connections → เลือก Integration ที่สร้าง
5
คัดลอก Database ID จาก URL: notion.so/xxxxxxxx...xxxxx?v=... เอาแค่ 32 ตัว
